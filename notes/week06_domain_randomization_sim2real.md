# Week06 Task D: MuJoCo Domain Randomization 与 Sim-to-Real 小实验

## 1. 实验目的

本实验用于理解机器人控制中的 sim-to-real gap、domain randomization、system identification 和 robustness。

实验基于 Week2 的 MuJoCo 单关节模型 `hinge_rod.xml`，固定 PD 控制器参数不变，只改变仿真环境中的质量、阻尼、关节摩擦和动作噪声，观察这些参数扰动对控制效果的影响。

本实验的核心问题是：

> 当仿真模型参数和真实机器人不完全一致时，同一个控制器还能不能稳定工作？

---

## 2. 核心概念

### Sim-to-Real Gap

Sim-to-real gap 指仿真环境和真实机器人之间的差距。

真实机器人中的质量、惯量、摩擦、阻尼、电机输出、传感器噪声和控制延迟通常无法被仿真完全准确建模，因此仿真中表现良好的控制器，在真实系统上可能出现性能下降。

### Domain Randomization

Domain randomization 是在仿真训练或测试过程中随机扰动物理参数，使控制器不要只适应单一理想环境，而是适应一组可能的环境分布。

其目标不是让仿真参数绝对准确，而是提升控制器面对参数不确定性时的鲁棒性。

### System Identification

System identification 是通过真实机器人实验数据反推系统参数，使仿真模型尽量接近真实系统。

例如，可以通过关节响应曲线估计质量、惯量、阻尼、摩擦和电机参数。

### Robustness

Robustness 指控制器在模型参数变化、噪声扰动或执行误差存在时，仍能保持稳定控制效果的能力。

---

## 3. 实验设置

### 模型与控制器

* 模型：`models/hinge_rod.xml`
* 控制器：固定 PD 控制器
* 目标角度：45 deg
* 控制形式：

```text
torque = Kp * error - Kd * theta_dot
```

其中：

* `Kp = 20`
* `Kd = 2`
* torque 被限制在 `[-5, 5]`

### 扰动场景

固定扰动实验包括：

| 场景                    | 含义              |
| --------------------- | --------------- |
| `nominal`             | 标准仿真环境          |
| `mass_x0.5`           | 杆质量变为原来的 0.5 倍  |
| `mass_x2.0`           | 杆质量变为原来的 2 倍    |
| `damping_x0.3`        | 关节阻尼变为原来的 0.3 倍 |
| `damping_x3.0`        | 关节阻尼变为原来的 3 倍   |
| `joint_friction_0.05` | 加入关节干摩擦         |
| `action_noise_0.5`    | 对动作力矩加入随机噪声     |

### 评价指标

实验记录以下指标：

* `final_angle_deg`：最终角度
* `final_error_deg`：最终角度误差
* `max_overshoot_deg`：最大超调
* `mean_abs_torque`：平均控制力矩绝对值
* `success`：是否满足脚本设定的成功条件

## 5. 结果分析

### 5.1 纯 PD 控制存在稳态误差

在 nominal 场景下，杆最终停在约 42.17 deg，距离目标 45 deg 仍有约 2.83 deg 的误差。

这是因为当前控制器是纯 PD 控制，没有积分项，也没有重力补偿。在存在重力负载的情况下，关节需要持续力矩来维持目标角度，而纯 PD 控制需要保留一定误差来产生该力矩。

因此，这里的稳态误差不是随机误差，而是控制器结构导致的现象。

### 5.2 质量变化对最终误差影响最大

质量变为 0.5 倍时，最终误差从 2.83 deg 降低到 1.45 deg，平均控制力矩也从 1.027 降低到 0.534。

质量变为 2 倍时，最终误差增大到 5.38 deg，并被判定为失败，平均控制力矩也增加到 1.906。

这说明在固定 PD 参数下，控制器对质量和惯量变化比较敏感。

从 sim-to-real 角度看，如果真实机器人连杆质量、负载或惯量和仿真模型不一致，控制器性能可能明显下降。

### 5.3 阻尼变化对最终误差影响不明显

`damping_x0.3` 和 `damping_x3.0` 的最终误差几乎与 nominal 相同。

这说明在当前实验指标下，阻尼变化没有明显影响最终静态误差。

但阻尼主要影响的是动态过程，例如响应速度、振荡和收敛过程。如果要更明显观察阻尼影响，可以进一步加入 rise time、settling time 等指标。

### 5.4 关节摩擦影响较小

`joint_friction_0.05` 的最终误差为 2.843 deg，与 nominal 的 2.830 deg 非常接近。

这说明当前摩擦扰动较小，或者 PD 控制器输出的力矩足以克服该摩擦，因此最终误差变化不明显。

### 5.5 动作噪声影响轨迹平滑性

`action_noise_0.5` 的最终误差为 2.932 deg，与 nominal 相比只略有变大。

但是从角度曲线可以观察到，`action_noise_0.5` 相比 nominal 曲线更加抖动。

这说明动作噪声虽然没有显著增大最终静态误差，但会降低控制过程的平滑性。这对应真实机器人中电机输出误差、通信抖动或执行器噪声对运动稳定性的影响。

## 6. Domain Randomization 结果分析

随机扰动实验中，质量、阻尼、摩擦和动作噪声在一定范围内随机采样。

从结果可以观察到，质量较大的 episode 往往最终角度更低，最终误差更大；质量较小的 episode 往往最终角度更接近目标。

例如：

* `domain_randomization_04` 中，`mass_scale = 1.795`，最终角度约为 40.07 deg，误差约为 4.93 deg。
* `domain_randomization_05` 中，`mass_scale = 0.542`，最终角度约为 43.39 deg，误差约为 1.61 deg。

这进一步说明，当前控制器对质量参数较敏感。

---

## 7. 实验结论

本实验说明，在固定 PD 控制器下，MuJoCo 单关节系统对质量变化最敏感。

当质量减小时，控制器更容易把杆控制到目标角度附近；当质量增大时，控制器需要更大的抗重力力矩，最终误差明显增大，甚至导致失败。

阻尼、较小关节摩擦和动作噪声在当前设置下对最终误差影响较弱，但动作噪声会使轨迹曲线更加抖动，降低控制过程的平滑性。

该实验展示了 sim-to-real gap 的一个简单来源：仿真中的质量、惯量、摩擦、阻尼和执行器输出如果与真实机器人不一致，控制器性能可能下降。

为了提高真实迁移能力，可以采用两类方法：

1. System Identification：通过真实机器人实验数据校准仿真参数，使仿真更接近真实系统。
2. Domain Randomization：在仿真中随机扰动物理参数，使控制器适应一组可能的真实环境，从而提升 robustness。

---

## 8. 面试表达

这个实验中，我用 MuJoCo 单关节模型做了一个最小 sim-to-real gap 实验。控制器固定为 PD 控制，只扰动环境参数，包括质量、阻尼、关节摩擦和动作噪声。

实验发现，质量变化对最终控制误差影响最大。质量变为 0.5 倍时，最终误差降低到约 1.45 deg；质量变为 2 倍时，最终误差增大到约 5.38 deg，并被判定为失败。阻尼和较小摩擦对最终误差影响较小，而动作噪声主要让角度曲线变得更抖。

这个实验说明，如果仿真中的质量、惯量和执行器模型与真实机器人不一致，控制器迁移到真实系统时可能出现性能下降。解决思路包括 system identification 和 domain randomization，前者让仿真更接近真实机器人，后者让策略适应一组可能的参数分布，最终目标都是提升控制器的 robustness。
