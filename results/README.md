# Results

本目录用于保存实验输出结果。

## 保存规范

每次正式实验至少保存以下内容：

- 配置文件：对应的 YAML 配置复制到实验结果目录；
- 随机种子：记录在配置文件或实验说明中；
- 指标数据：CSV 格式；
- 视频路径：记录评估视频或 GIF 的保存位置；
- 实验结论：简要说明该次实验的目的与结果。

## 目录说明

- `logs/`：训练日志与 TensorBoard 文件，不直接提交大量原始文件；
- `videos/`：评估视频文件，正式展示材料可筛选后放入 `assets/`；
- `checkpoints/`：模型权重文件，通常不直接提交到 Git。
